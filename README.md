Reprint is a simple tool that converts PDF documents to EPUB format using Azure Document Intelligence (formerly Form Recognizer), with support for preserving tables and hyperlinks.

I hope to evolve this project into a module that could be combined with a simple API layer to implement an online service to facilitate my own conversion needs for reMarkable in the future.

## Why EPUB?

While PDFs are great for preserving exact document layouts, they can be cumbersome to read on e-readers and mobile devices. EPUB files offer several advantages:

- **Reflowable text** that automatically adjusts to your screen size and preferred font settings
- **Better reading experience** on e-readers like Kindle, Kobo, and reMarkable tablets
- **Smaller file sizes** compared to PDFs
- **Enhanced accessibility** with support for text-to-speech and customizable fonts

## Getting Started

This project uses VSCode devcontainers for development. To get started:

1. Install [VSCode](https://code.visualstudio.com/) and the [Dev Containers extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)
2. Clone this repository
3. Open in VS Code and click "Reopen in Container" when prompted
